# FZHDrawCircle
一个绘制圆形统计图的视图
====
## 1、需要设置的属性<br>
  * `colourFillColor`：填充颜色<br>
  * `colourStartAngle `：起始位置<br>
  * `colourEndAngle`：结束位置<br>
  * `normalFillColor`：填充颜色<br>

## 2、可选设置的属性<br>
  * cateLabel：所属类别<br>
  * moneyNumLabel：花费金额<br>
